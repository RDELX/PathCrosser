# PathCrosser

A World of Warcraft addon for tracking player encounters, coordinates, and analytics.

## Features

- **Encounter Tracking**: Records when and where you meet other players.
- **Player Details**: View encountering history, level, class, race, and guild.
- **Relation System**: Mark players as friends, rivals, or neutral with custom tags.
- **Statistics**: Shows most encountered players, active zones, and class distributions.
- **Tooltip Integration**: Hover over players to quickly see encounter count, relations, and last known location.

## Installation

1. Download the latest release.
2. Extract the `PathCrosser` folder to your WoW AddOns directory:
   - `World of Warcraft\_retail_\Interface\AddOns\`
3. Restart WoW or reload your UI (`/reload`).

### Commands

| Command | Description |
|---------|-------------|
| `/pc` or `/pathcrosser` | Open main window |
| `/pc stats` | Show quick statistics in chat |
| `/pc prune` | Manually prune old encounters |
| `/pc help` | Display all available commands |
